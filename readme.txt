This plugin allows you to use a shortcode to display anywhere in WordPress HTML sitemap with posts and pages list. You can also change title of post and pages sections.

== Description ==

This plugin allows you to display anywhere in WordPress generated HTML sitemap with all pages and posts. HTML sitemap can be easily displayed anywhere in WordPress without modifying the code and it works with most of the available themes. Just install and activate this plugin and use [sp_sitemap_html] shortcode anywhere in WordPress to display generated HTML sitemap with posts and pages list. Also go to options pages Sitemap HTML to change the title of pages and posts sections.<br><br>Read more about this plugin on <a href="https://siriuspro.pl">WordPress</a> blog.

== Installation ==

1. Unzip the wordpress-html-sitemap-generator.zip folder.
2. Upload the wordpress-html-sitemap-generator folder to your /wp-content/plugins directory.
3. In your WordPress dashboard, head over to the Plugins section.
4. Activate Sitemap HTML Generator plugin.
5. Go to options and then put [sp_sitemap_html] shortcode anywhere in WordPress.
